# Classification of Crocs Purchase Intention Using Artificial Neural Networks

## Overview

This project aims to classify customers' purchase intention toward Crocs products using an Artificial Neural Network (ANN). Two activation functions, Sigmoid and ReLU, are compared to evaluate model performance.

## Project Files

- 📓 **Crocs_ANN_Analysis.ipynb** – Complete notebook containing data preprocessing, ANN model development, training, and evaluation.
- 📊 **Crocs_Survey.xlsx** – Survey dataset used for the analysis.
- 📄 **Classification_of_Crocs_Purchase_Intention.pdf** – Research paper describing the methodology and findings.

## Tools & Libraries

- Python
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- Matplotlib

## Objective

Develop an Artificial Neural Network model to predict Crocs purchase intention and compare the performance of Sigmoid and ReLU activation functions.
